**MyTool** is a high-performance, multi-utility Android application designed with a retro-terminal (Hacker) aesthetic. It combines essential web development tools, security features, and a high-speed secure browser into a single, lightweight package.

---

## 🛠 CORE_FEATURES

### 1. 🌐 Secure Browser
- **High Speed:** Optimized for rapid page loading.
- **Ad-Blocker:** Built-in logic to block common intrusive ads.
- **Desktop Mode:** Toggle between mobile and desktop views with one click.
- **Save Page:** Securely archive your favorite URLs for offline access.

### 2. 💻 Data Validation Lab (Live Editor)
- **Real-time Preview:** Write HTML, CSS, and JS and see instant results.
- **Multi-Action Bar:** Features for **Clear**, **Copy**, and **Run in External Browser**.
- **Save Snippets:** Save your code directly to the system archive.
- **Toggle View:** Seamlessly switch between the code editor and live preview.

### 3. 🔐 Secure Encryptor & Smart Bar
- **Text Obfuscation:** Base64 based encryption/decryption.
- **Smart URL Search:** Auto-detects URLs or performs Google searches directly from the input.
- **Dynamic Buttons:** Buttons morph between Encrypt/Decrypt and Enter/Search based on input.

### 4. 🔑 Password Lab
- **Unhackable Keys:** Generates strong 16-character passwords.
- **Entropy-Rich:** Uses uppercase, lowercase, numbers, and symbols.
- **Quick Copy:** One-tap copy to system clipboard.

### 5. 📡 Network Monitor
- **Link Speed:** Real-time monitoring of Download and Upload bandwidth.
- **Console Logs:** Results are logged directly into the dashboard terminal.
